Android Engineering Skill Pkg es un paquete open source que encapsula conocimientos de ingeniería Android en un formato reutilizable para asistentes de IA. Define patrones, mejores prácticas y arquitecturas de referencia que cualquier agente de IA puede consumir para generar código Android de alta calidad. Incluye guías detalladas de Clean Architecture, patrones de diseño específicos de Android, configuración de Gradle optimizada, estrategias de testing y gestión de dependencias. Publicado como paquete NPM para facilitar su integración en cualquier flujo de trabajo.
