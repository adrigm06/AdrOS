<Content lang="es">
BetterPrompt es una herramienta de terminal (TUI) diseñada para optimizar la creación y refinamiento de prompts para modelos de lenguaje. Utiliza análisis heurístico para evaluar la calidad de los prompts, sugiriendo mejoras estructurales y semánticas. Construida con TypeScript y Node.js, ofrece una interfaz limpia y eficiente para ingenieros de prompts que prefieren trabajar desde la terminal. Entre sus características destacan: análisis de claridad y especificidad del prompt, detección de ambigüedades, sugerencias de contexto adicional, y exportación a múltiples formatos.
</Content>

<Content lang="en">
BetterPrompt is a terminal (TUI) tool designed to optimize the creation and refinement of prompts for language models. It uses heuristic analysis to evaluate prompt quality, suggesting structural and semantic improvements. Built with TypeScript and Node.js, it offers a clean and efficient interface for prompt engineers who prefer working from the terminal. Features include: clarity and specificity analysis, ambiguity detection, additional context suggestions, and multi-format export.
</Content>
