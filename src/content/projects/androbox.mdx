<Content lang="es">
AndroBox es un entorno de trabajo todo-en-uno para desarrolladores Android. Combina un editor de código ligero, gestor de proyectos, integración con ADB y herramientas de profiling en una sola interfaz. Diseñado para agilizar el día a día del desarrollador Android, AndroBox integra snippets inteligentes, gestión de dependencias y previsualización en tiempo real de layouts Compose. El proyecto está construido con Kotlin y Jetpack Compose, siguiendo principios de Clean Architecture para garantizar mantenibilidad y testabilidad.
</Content>

<Content lang="en">
AndroBox is an all-in-one workbench for Android developers. It combines a lightweight code editor, project manager, ADB integration, and profiling tools in a single interface. Designed to streamline the Android developer's daily workflow, AndroBox integrates smart snippets, dependency management, and real-time Compose layout preview. The project is built with Kotlin and Jetpack Compose, following Clean Architecture principles to ensure maintainability and testability.
</Content>
