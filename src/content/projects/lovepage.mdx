<Content lang="es">

LovePage es una aplicación web full-stack diseñada para crear un espacio digital personalizado y afectivo. Frontend dinámico con HTML5, CSS3 y JavaScript modular; backend con Node.js, Express y PostgreSQL. Desplegada en GitHub Pages + Render.

> **Demo en vivo:** [adrigm06.github.io/LovePage](https://adrigm06.github.io/LovePage/)

#### Funcionalidades

| Categoría | Características |
|---|---|
| **Tema visual** | Modo día/noche con burbujas animadas (día) y cielo estrellado (noche) |
| **Animaciones** | Corazones flotantes, máquina de escribir, transiciones suaves |
| **Cursor** | Personalizado con forma de corazón |
| **Mensajes** | Sistema con efecto de escritura y latidos, personalizables por usuario |
| **Galería** | Fotos compartidas con efectos 3D y rotación automática basada en fechas |
| **Contadores** | Días desde la fecha especial + badges mensuales (🍊 mandarinas) |
| **Fechas** | Gestión de eventos importantes con iconos personalizables |
| **Spotify** | Reproductor embebido con playlist personalizable |
| **Autenticación** | Login/registro con sesiones, perfiles y ajustes por usuario |

#### Tecnologías

**Frontend:**
- HTML5, CSS3 modular (variables, temas, animaciones, galería, formularios)
- JavaScript vanilla con arquitectura modular (config, animaciones, temas, contadores, mensajes, galería, Spotify, fotos, login, recordatorios)

**Backend:**
- Node.js + Express, sesiones con express-session, API RESTful con CORS
- PostgreSQL en la nube, dotenv para configuración

**Despliegue:**
- Frontend: GitHub Pages
- Backend + DB: Render

#### Estructura del Proyecto


<Content lang="en">

LovePage is a full-stack web app designed to create a personalised, heartfelt digital space. Dynamic frontend with HTML5, CSS3 and modular JavaScript; backend with Node.js, Express and PostgreSQL. Deployed on GitHub Pages + Render.

> **Live demo:** [adrigm06.github.io/LovePage](https://adrigm06.github.io/LovePage/)

#### Features

| Category | Details |
|---|---|
| **Visual theme** | Day/night toggle with animated bubbles (day) and starry sky (night) |
| **Animations** | Floating hearts, typewriter effect, smooth transitions |
| **Cursor** | Custom heart-shaped cursor |
| **Messages** | Typewriter system with heartbeat, customisable per user |
| **Gallery** | Shared photos with 3D effects and date-based rotation |
| **Counters** | Days since special date + monthly badges (🍊 mandarins) |
| **Dates** | Important event management with customisable icons |
| **Spotify** | Embedded player with customisable playlist |
| **Auth** | Login/register with sessions, profiles and per-user settings |

#### Tech Stack

**Frontend:**
- HTML5, modular CSS3 (variables, themes, animations, gallery, forms)
- Vanilla JavaScript with modular architecture (11 modules)

**Backend:**
- Node.js + Express, express-session, RESTful API with CORS
- Cloud PostgreSQL, dotenv configuration

**Deployment:**
- Frontend: GitHub Pages
- Backend + DB: Render
