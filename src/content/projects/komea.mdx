<Content lang="es">

Komea es una aplicación Android nativa —actualmente en fase avanzada de desarrollo y próxima a lanzarse a producción— que optimiza la gestión alimentaria mediante **Inteligencia Artificial Generativa**. Unifica la planificación de menús y el control de inventario doméstico en una única experiencia automatizada.

#### Arquitectura

| Capa | Enfoque |
|---|---|
| **UI** | Jetpack Compose + Material Design 3 |
| **Arquitectura** | Clean Architecture + MVVM + UDF (flujo unidireccional) |
| **IA** | Procesamiento y estructuración de datos con IA Generativa |
| **Lenguaje** | Kotlin 100% nativo |
| **Escalabilidad** | Diseño modular orientado a producto |

#### Funcionalidades Principales

| Característica | Descripción |
|---|---|
| 🧠 **Planificación inteligente** | La IA genera menús personalizados según preferencias, restricciones y disponibilidad |
| 📋 **Lista de la compra automática** | Generada directamente desde la planificación semanal |
| 🥘 **Recetas personalizadas** | Sugerencias basadas en ingredientes disponibles en casa |
| 📷 **Escaneo de alimentos** | Reconocimiento visual para registrar inventario |
| 👥 **Comunidad de recetas** | Comparte recetas, la comunidad las adapta y genera nuevas versiones |
| 🔄 **Adaptación en tiempo real** | El sistema se ajusta dinámicamente a cambios en la nevera o preferencias |

#### Impacto

- **Generación automática** de listas de la compra
- **Reducción del esfuerzo manual** en la planificación diaria
- **Ahorro de alimentos** al priorizar ingredientes disponibles
- **Creatividad culinaria colaborativa** con recetas compartidas

> Proyecto en desarrollo avanzado — lanzamiento público previsto próximamente.

<Content lang="en">

Komea is a native Android application —currently in advanced development and nearing production launch— that optimises food management through **Generative AI**. It unifies meal planning and household inventory control into a single automated experience.

#### Architecture

| Layer | Approach |
|---|---|
| **UI** | Jetpack Compose + Material Design 3 |
| **Architecture** | Clean Architecture + MVVM + UDF |
| **AI** | Generative AI for data processing and structuring |
| **Language** | 100% native Kotlin |
| **Scalability** | Product-oriented modular design |

#### Key Features

| Feature | Description |
|---|---|
| 🧠 **Smart planning** | AI generates personalised menus based on preferences, restrictions and availability |
| 📋 **Auto shopping list** | Generated directly from weekly meal plan |
| 🥘 **Personalised recipes** | Suggestions based on ingredients at home |
| 📷 **Food scanning** | Visual recognition to log inventory |
| 👥 **Recipe community** | Share recipes, community adapts and generates new versions |
| 🔄 **Real-time adaptation** | System adjusts dynamically to fridge changes or preferences |

#### Impact

- **Automatic** shopping list generation
- **Reduced manual effort** in daily meal planning
- **Food waste reduction** by prioritising available ingredients
- **Collaborative culinary creativity** with shared recipes

> Project in advanced development — public launch coming soon.
