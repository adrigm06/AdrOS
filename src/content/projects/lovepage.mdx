<Content lang="es">
LovePage es un constructor de landing pages románticas que permite a los usuarios crear páginas web personalizadas para declaraciones de amor, aniversarios o propuestas. El frontend estático está construido con HTML5, CSS3 y JavaScript vanilla, desplegado en GitHub Pages. El backend con Node.js, Express.js y PostgreSQL gestiona la autenticación, almacenamiento de configuraciones y entrega de páginas. Incluye animaciones CSS, música de fondo personalizable y un generador de contenido romántico.
</Content>

<Content lang="en">
LovePage is a romantic landing page builder that lets users create personalized web pages for love confessions, anniversaries, or proposals. The static frontend is built with HTML5, CSS3, and vanilla JavaScript, deployed on GitHub Pages. The backend with Node.js, Express.js, and PostgreSQL handles authentication, configuration storage, and page delivery. Includes CSS animations, customizable background music, and a romantic content generator.
</Content>
