<Content lang="es">

PetConnect es una plataforma integral para la digitalización del sector veterinario, desarrollada como monorepo con backend Spring Boot, frontend React + TypeScript y versión móvil con Capacitor/Android.

#### Stack Tecnológico

| Capa | Tecnologías |
|---|---|
| **Backend** | Java 21, Spring Boot 3.4.1, Spring Security (JWT) |
| **Base de datos** | PostgreSQL vía Supabase, JPA / Hibernate |
| **Frontend** | React 19, TypeScript, Vite 8, TailwindCSS 3 |
| **Estado** | Zustand |
| **Mapas** | Leaflet + React Leaflet + Overpass API |
| **Móvil** | Capacitor (Android) |
| **Auth** | Supabase Auth + JWT Resource Server |

#### Mapa Interactivo

El mapa está centrado en España con las siguientes categorías de centros veterinarios:

| Categoría | Descripción |
|---|---|
| 🏥 Clínicas veterinarias | Centros de atención primaria |
| 🚑 Urgencias 24h | Veterinarios de emergencia |
| 🏨 Hospitales 24h | Hospitales veterinarios con internamiento |
| 🏠 Refugios y protectoras | Centros de adopción y acogida |
| 🔍 Centros de acogida | Animales perdidos temporalmente |
| 🛒 Tiendas de animales | Categoría secundaria |

**Estrategia de carga del mapa:**
- Caché en memoria + persistente en localStorage
- Reintentos con backoff ante errores 429/504 de Overpass API
- Reducción de concurrencia para evitar rate limiting
- Fallback automático a datos demo sin conexión

#### Red Social para Propietarios

- Publicaciones con imágenes y vídeo (multipart/form-data)
- Sistema de likes y comentarios por post
- Seguimiento entre usuarios (follow / unfollow)
- Búsqueda y paginación en backend para escalar
- Perfiles médicos de mascotas compartibles

<Content lang="en">

PetConnect is a comprehensive platform for veterinary sector digitalization, built as a monorepo with a Spring Boot backend, React + TypeScript frontend, and Capacitor/Android mobile support.

#### Tech Stack

| Layer | Technologies |
|---|---|
| **Backend** | Java 21, Spring Boot 3.4.1, Spring Security (JWT) |
| **Database** | PostgreSQL via Supabase, JPA / Hibernate |
| **Frontend** | React 19, TypeScript, Vite 8, TailwindCSS 3 |
| **State** | Zustand |
| **Maps** | Leaflet + React Leaflet + Overpass API |
| **Mobile** | Capacitor (Android) |
| **Auth** | Supabase Auth + JWT Resource Server |

#### Interactive Map

The map is centered on Spain with the following veterinary center categories:

| Category | Description |
|---|---|
| 🏥 Veterinary clinics | Primary care centers |
| 🚑 24h Emergencies | Emergency veterinary services |
| 🏨 24h Hospitals | Veterinary hospitals with inpatient care |
| 🏠 Shelters | Adoption and foster centers |
| 🔍 Lost animal centers | Temporary lost pet holding |
| 🛒 Pet stores | Secondary category |

**Map loading strategy:**
- In-memory cache + persistent localStorage cache
- Backoff retries on Overpass API 429/504 errors
- Concurrency reduction to prevent rate limiting
- Automatic fallback to demo data offline

#### Social Network for Pet Owners

- Posts with image and video support (multipart/form-data)
- Like and comment system per post
- User follow / unfollow
- Search and pagination on the backend
- Shareable pet medical profiles
