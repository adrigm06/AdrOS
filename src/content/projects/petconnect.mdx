<Content lang="es">
PetConnect es una plataforma full-stack que conecta refugios de animales con adoptantes potenciales. Desarrollada con Java 21 y Spring Boot 3.4 en el backend, y React con TypeScript en el frontend. Utiliza Spring Security con JWT para autenticación, PostgreSQL con Supabase para almacenamiento, y Leaflet para mapas interactivos que muestran mascotas cercanas. Los refugios pueden gestionar perfiles de animales, y los adoptantes pueden filtrar por especie, edad, ubicación y compatibilidad. Incluye un sistema de mensajería integrado y seguimiento post-adopción.
</Content>

<Content lang="en">
PetConnect is a full-stack platform connecting animal shelters with potential adopters. Built with Java 21 and Spring Boot 3.4 on the backend, and React with TypeScript on the frontend. Uses Spring Security with JWT for authentication, PostgreSQL with Supabase for storage, and Leaflet for interactive maps showing nearby pets. Shelters can manage animal profiles, and adopters can filter by species, age, location, and compatibility. Includes an integrated messaging system and post-adoption tracking.
</Content>
